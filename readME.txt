Alexa.py
******
A tiny python script that downloads and outputs the top visited websites in the world from Alexa.

This script uses Request lib to download the page and lxml lib to extract the content from HTML. Finally the script also uses pandas lib to output the data in a nicely formatted format.


Saruk_digital.py
******
A tiny python script that scrapes laptop prices from on of the Kenyan leading laptop sellers.

This script uses Request lib to download the page and lxml lib to extract the content from HTML. Finally the script also uses pandas lib to output the data in a nicely formatted format.
